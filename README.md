# neural-dialog
this project is a storage for important files of science paper

Этот проект реализует систему голосового взаимодействия между двумя независимыми AI-агентами.
Код предназначен для запуска на двух разных устройствах (например, смартфонах через Google Colab), которые «слушают» друг друга и ведут осмысленный диалог в реальном времени.

## Сценарий использования

1.  Код запускается на двух телефонах/компьютерах, расположенных рядом.
2.  Каждому агенту можно задать уникальный `system_prompt`.
3.  Голосовой цикл:
    * Агент А произносит начальную фразу.
    * Агент Б через микрофон ловит звук, распознает его (Whisper), генерирует ответ (Qwen 2.5) и озвучивает его (Edge-TTS).
    * Агент А слышит ответ и реагирует.
    * Цикл продолжается до достижения «триггера согласия» или ручной остановки.

## Технологический стек

* Модель генерации текста: `Qwen/Qwen2.5-3B-Instruct`.
* Модель распознавания речи `OpenAI Whisper (small)`.
* Модель смнтеза речи `edge-tts`.

## Настройка и запуск

### 1. Подготовка (на обоих устройствах)
* Откройте ноутбуки в Google Colab.
* Выполните первуые ячейку для установки всех зависимостей.

### 2. Настройка промптов
Измените функцию `generate_ai_response` на каждом устройстве:
* Устройство 1: задаём "content"
* Устройство 2: задаём "content"

### 3. Запуск диалога
1.  Запустите код на втором устройстве
2.  Нажмите кнопку «Включить микрофон» на втором устройстве.
3.  Запустите код на первом устройстве
4.  Положите устройства рядом.
